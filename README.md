# VERRE Interactive Progressive Lens Application

## Introduction
VERRE is an interactive application designed for exploring and understanding progressive lenses. Whether you're a user, an optician, or simply curious, VERRE provides valuable insights into the design and function of progressive lenses.

## Features
- **Interactive Simulations**: Experience how different lens designs impact vision.
- **Comparative Tools**: Compare different types of progressive lenses side by side.
- **User Guides**: Access resources on how to choose the right lens and care for them.

## Getting Started
To get started with the VERRE application, please follow these steps:
1. Clone the repository: `git clone https://github.com/steven77390/VERRE`
2. Navigate into the project directory: `cd VERRE`
3. Install dependencies: `npm install`
4. Run the application: `npm start`
